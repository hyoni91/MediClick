<h1>🩺 MediClick PROJECT</h1>
<br/>
<span>
MediClick은 암센터 웹 사이트를 구현한 프로젝트입니다.<br/>
환자와 관리자가 편하게 서비스를 이용할 수 있도록 설계했습니다.<br/?
    
본 프로젝트를 진행하며 Java, JavaScript, React, Maria DB, Spring, Mybatis 등<br/>
다양한 프로그래밍 언어뿐만 아니라 오픈API와 다양한 라이브러리 등을 활용하여<br/>
사용자에게 편리함을 줄 수 있는 웹 사이트를 만들기 위해 노력했습니다.</span>
<br/>
